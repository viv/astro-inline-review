# Astro Inline Review -- Example

Minimal Astro project for manually testing the annotation overlay.

## Setup

```sh
npm install
npm run dev
```

Open http://localhost:4321 in your browser.

## What to test

- **Text annotation**: select any text on the page and add a note.
- **Element annotation**: Alt+click a card, button, or link.
- **Panel**: click the floating action button (bottom-right) to open the annotations panel, review and delete annotations.
- **Scroll dismiss**: open a popup, then scroll -- it should close after ~50px.
