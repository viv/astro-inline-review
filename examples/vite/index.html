This page exists to manually test the annotation overlay provided by astro-inline-review when used as a Vite plugin.

Select any portion of this paragraph to test text annotation. The overlay should appear after you release the mouse button, allowing you to attach a note to the selected text. Try selecting a single word, a phrase, or an entire sentence to verify different selection sizes work correctly.
